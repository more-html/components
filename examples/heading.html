More HTML - Heading components

When hovering headline above an icon shows up, with a link directly to the anchor of this tag. This allows for direct links to headlines as anchors.

  1. Click the link at the end of the headline, that appears on mouseover
  2. See the URL change, a #something is added to the end
  3. You can copy+paste that link and it leads straight to this headline

See the examples page, with different configurations and settings and contents.
